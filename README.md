## 01:pdf-online-viewer

### 简介
移动端在线预览pdf文件的一个小案例，使用了第三方库 `PDFJs`，并且添加了移动端左右上下滑动切换页面的效果。

### 用法
本例需要运行在服务器环境下，直接以文件形式在浏览器打开不会有效果。建议使用 `http-server` :
```
npm install http-server
````
然后切换到项目的根目录：

```
http-server ./
```
直接打开链接即可
